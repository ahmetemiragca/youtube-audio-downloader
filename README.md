# YouTube Audio Downloader

## Description:
This project is a fork of [yt-dlp](https://github.com/yt-dlp/yt-dlp). It aims to make it easier to download audio of YouTube content. Please follow COPYRIGHT rules. This project has been developed for hobby purposes.
